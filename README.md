# 📊 Salary Insights Project

## 🔎 Overview
This project demonstrates an end‑to‑end data analytics workflow using **Python**, **Power BI**, and a supporting **PPT presentation**.  
Starting from raw data, I cleaned, explored, visualized, and generated actionable insights — all presented in a recruiter‑ready format.

---

## 🛠️ Tools & Technologies
- **Python**: pandas, NumPy, Matplotlib, Seaborn  
- **Jupyter Notebooks**: modular workflow for analysis  
- **Power BI**: interactive dashboard with slicers and KPIs  
- **GitHub**: version control & portfolio presentation  
- **PPT Presentation**: concise storytelling of insights  

---

## 📂 Project Structure
- **1_Data_Pre-processing.ipynb** → Cleaning, handling missing values, feature engineering  
- **2_EDA.ipynb** → Exploratory Data Analysis (patterns, distributions, correlations)  
- **3_Visualizations.ipynb** → Charts and plots for salary trends  
- **4_Insights.ipynb** → Key findings and recommendations  
- **5_Complete_Salary_Analysis_Project.ipynb** → Final polished report notebook  
- **Salary Analysis.pbix** → Interactive dashboard for recruiter engagement  
- **Presentation.pptx** → Slide deck summarizing workflow, insights, and recommendations  

---

## 📈 Key Insights
- Salaries rise steadily with **experience**.  
- **Remote work** shows a slight premium compared to hybrid/on‑site.  
- **Enterprise companies** consistently pay more than startups.  
- **Certifications** add incremental salary growth.  
- **Location** has the strongest impact — USA highest, India lowest.
